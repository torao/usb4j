ClassID XML 変換ユーティリティ

目 的

このユーティリティは USB ID's ファイルを usb4j の ClassID クラスで読み込み可能な形式の XML に 変換します。ClassID XML はクラス、サブクラス、ベンダー、製品などの ID を人が認識するためのラベルとマップするための 簡単なデータベースファイルです。USB ID's ファイルは http://www.linux-usb.org などで入手することが出来ます。

利用方法

IDS ファイルの URL またはローカルファイルを選択して変換ボタンを押してください。変換内容を確認したら保存ボタンでファイルに 保存することが出来ます。ClassID XML ファイルはリソースとして JAR ファイルにバンドルしアプリケーションで使用する事が できます。 

String resourceName = "com/foo/usbclassid.xml";
URL url = classLoader.getResource(resourceName);
ClassID classId = ClassID.getClassID(url);
String vendorName = classId.getVencor(0x8086);
